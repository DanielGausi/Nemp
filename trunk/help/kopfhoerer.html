Abspielen im Kopfhörer

Mit Nemp können Sie zwei Dateien parallel abspielen. Wenn Sie zwei Soundkatren in Ihrem Rechner eingebaut haben, können Sie an den Ausgang der zweiten Karten einen Kopfhörer anschließen, und darüber ein anderes Lied hören als über die Boxen.
Gedacht ist das für den Einsatz auf einer Party, wenn man in ein anderes Lied reinhören möchte, ohne die Hauptausgabe zu stören.

Wenn nur eine Soundkarte installiert ist, oder für die Hauptausgabe und die Kopfhörerausgabe dieselbe Soundkarte ausgewählt ist, werden beide Titel auf einem Ausgabegerät abgespielt.

Markieren Sie dazu einfach einen Titel in der Playlist oder der Medienbibliothek und wählen übder das Kontextmenü "Im Kopfhörer abspielen". Es öffnet sich dann ein kleines Zusatzfenster, über das Sie die zweite Wiedergabe rudimentär steuern können.
