Medienbibliothek

Verzeichnisse
Nemp kann einige Verzeichnisse beim Start nach neuen Dateien untersuchen, um die Medienbibliothek automatisch aktuell zu halten, wenn Sie neue Musik auf Ihren Rechner kopiert haben. Sie können hier neue Verzeichnisse hinzufügen oder vorhandene entfernen.

Hinweis: Nemp verhindert Doppeleinträge. D.h. wenn Sie ein Unterverzeichnis eines anderen angeben, das schon in der Liste ist, wird das neue nicht eingeführt. Umgekehrt wird das vorhandene entfernt, wenn Sie ein übergeordnetes Verzeichnis angeben.

  • Automatisch neue Verzeichnisse überwachen fügt ein Verzeichnis in diese Liste ein, wenn Sie es per Drag&Drop in die Medienbibliothek einfügen oder über das Menü neue Ordner untersuchen.
  • Bei neuen Verzeichnissen nachfragen bewirkt, dass das erst auf Nachfragee passiert.

Dateitypen für die Medienbibliothek
Nemp kann zwar ziemlich viele Dateien abspielen, aber vernünftig untersuchen kann es nur wenige Typen. Sie können daher angeben, dass beim Scannen nach neuen Dateien z.B. wav- oder midi-Dateien nicht in die Medienbibliothek aufgenommen werden sollen.


Sonstiges

  • Dateien in Wiedergabelisten untersuchen: Wenn Sie in der klassischen Browse-Ansicht in der linken Liste den Punkt "Wiedergabelisten" auswählen, werden rechts daneben alle Playlisten aufgelistet, die Nemp gefunden hat. Wenn Sie davon eine auswählen, werden die Dateien dieser Playlist in der Medienliste unten angezeigt. Sie können angeben, ob Sie diese Dateien vor der Anzeige scannen lassen wollen, damit z.B. die Bitrate auch korrekt angezeigt wird, oder nicht.
  • Anzeige immer sortieren: Beim Browsen in der Medienbibliothek werden unten immer wieder neue Dateien angezeigt. Sie können auswählen, ob diese Liste immer so sortiert werden soll, wie Sie es vorher eingestellt haben. Um ein flüssiges Browsen zu ermöglichen, kann dieser Sortiervorgang bei langen Listen übersprungen werden.
