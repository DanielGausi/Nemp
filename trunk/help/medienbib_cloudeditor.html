Der Tagwolken-Editor

Wenn Sie sich über LastFM zusätzliche Tags für ihre Medienbibliothek besorgen, kann es vorkommen, dass für Sie völlig sinnlose Tags dazu kommen, oder einige Tags in verschiedenen Schreibweisen.

Mit dem Tagwolken-Editor können Sie damit etwas aufräumen.

Existierende Tags
Eine Auflistung aller, wirklich aller Tags in der Medienbibliothek, von häufigen wie "Singer-Songwriter" bis hin zu exotischen wie "songs die so gut sind das ich meiner oma ihr klein haeuschen zwar nicht verkaufen aber zumindest dafuer beleihen wuerde".
Sie können hier Tags umbenennen, zusammenführen oder löschen. Markieren Sie dazu einen oder mehrere Tags und klicken Sie den entsprechenden Button.
Die Option "Verstecke Auto-Tags" sorgt dafür, dass die Tags, die Nemp von sich aus erstellt hat (Interpret, Album, Jahr, Genre, Jahrzehnt) ausgeblendet werden. Diese Tags können Sie nicht ganz verändern. Sie können aber diese AutoTags für Dateien verändern, für die Nemp diesen Tag nicht selbst hinzugefügt hat.
Die Option "Speichere zusammengeführte/umbenannte/gelöschte Tags" sorgt dafür, dass Ihre Eingaben gespeichert werden und bei künftigen Abfragen an LastFM berücksichtigt werden.

Beispiel: Sie führen die beiden Tags "Female Vocalist" (ohne s am Ende) und "Female Vocalists" (mit s am Ende) zu "Female Vocalists" zusammen. Wenn sie später für ein Lied Tags von LastFM holen, und dieses Lied ist mit "Female Vocalist" (ohne s) getaggt, wird dieser Tag automatisch zu "Female Vocalists" (mit s) geändert.


Zusammengeführte/umbenannte Tags
In diesem Tab werden die Tags angezeigt, die sie zuvor umbenannt oder zusammengeführt haben. Diese Liste wird beim Holen neuer Tags von LastFM berücksichtigt, sodass die Tagwolke dadurch nicht wieder aufs neue zugemüllt wird.


Gelöschte Tags
Tags, die sie nicht in Ihrer Tagwolke sehen wollen. Auch diese Liste wird beim Holen neuer Tags berücksichtigt.


Dateien aktualisieren
Wenn Sie Tags umbenannt, zusammengeführt oder gelöscht haben, sollten Sie im Anschluss auch die Information in den mp3-Dateien selbst ändern lassen. Dieser Vorgang dauert eine Weile, in der Regel ein wenig länger als das Einlesen der Daten in die Medienbibliothek am Anfang.


Bugfix
In einer früheren Testversion von Nemp 4 war ein blöder Fehler drin, der dazu führte, dass die Tags nicht richtig in die Datei geschrieben wurden. In dieser Version von Nemp ist daher eine Funktion eingebaut, die diesen Fehler in den Dateien korrigieren kann.
Wenn Sie Nemp 4 zum ersten Mal benutzen, und keine Pre-Alpha-Version getestet haben, können Sie diese Funktion getrost ignorieren.
