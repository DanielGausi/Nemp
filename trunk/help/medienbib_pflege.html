Pflege der Medienbibliothek

Für eine korrekte Sortierung und ein schnelles Finden der Dateien ist es wichtig, dass die einzelnen Audiodateien mit ordentlichen Metadaten versehen sind.

Nemp 4.6 unterstützt folgende Systeme:
  • ID3-Tags (Version 1, 1.1, 2.2, 2.3 und 2.4)
  • Ogg-Vorbis-Kommentare
  • Flac-Metadaten
  • Apev2-Tags
  • iTunes Tags
Damit unterstützt Nemp Metatags unter anderen in *.mp3, *.ogg, *.flac, *.ape, *.mpc, *.ofr, *.tta, *.wv, *.m4a.
Nicht vollständig unterstützt werden unter anderen *.wma.


Informationen bearbeiten
Tipp 1: Klicken Sie (langsam) zweimal auf eine Information in der Titelanzeige der Medienbibliothek, oder drücken Sie dort F2
Tipp 2: Klicken Sie auf eine Information in der etwas detaillierteren Anzeige neben der Titelanzeige


Detailliertere Bearbeitung
Wenn Sie mehr wollen, können Sie das über das Kontextmenü "Eigenschaften" machen. Dann öffnet sich ein weiteres Fenster, in dem Sie die ID3Tags bearbeiten können und z.B. auch Cover und Liedtexte einfügen können.
Tipp: Versuchen Sie das Tastaturkürzel Strg+D. (D wie Details)


Liedtexte
Nemp kann zu einem Lied auch den Liedtext speichern, so dass man über die Suche auch ein Lied finden kann, das irgendwie so "in the town, where i was born" geht. Dieser Liedtext kann entweder von Hand eingegeben werden, oder automatisch aus dem Internet besorgt werden. Hierzu wird die Seite http://lyrics.wikia.com/ benutzt.
Markieren Sie einfach ein paar Dateien und wählen über das Kontextmenü "Lyrics holen".


Zusätzliche Tags
Nemp kann zu einem Titel (fast) beliebig viele weitere Tags speichern, die nicht in das übliche Schema "Interpret, Titel, Album, Genre" passen. Diese Tags können individuell gesetzt werden, oder automatisiert von last.fm übernommen werden. Mehr dazu bei der Tagwolke.


Aktualisieren der Daten
Wenn Sie einzelne Dateien mit anderen Programmen geändert haben, dann können Sie ein neues Einlesen für diesse Dateien erreichen.
Wählen Sie dazu über das Kontextmenü "Markierte aktualisieren" oder "Bibliothek neu einlesen".


Löschen von Dateien
Zum Löschen von Dateien aus der Medienbibliothek wählen Sie "Markierte Dateien aus der Bibliothek entfernen". Damit werden die Dateien aus der Bibliothek gelöscht, aber nicht von der Festplatte.
