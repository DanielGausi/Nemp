Anzeigen / Steuerung des Players



Titelanzeige
Tipp: Schalten Sie per Klick um zwischen verschiedenen Anzeigen:
  • Anzeige von Interpret - Titel, so wie es auch die Playliste macht
  • Anzeige des Dateinamens inklusive des komplette Pfades
  • Zusätzliche Infos zur Datei wie Bitrate, Samplerate und Dauer
  • Liedtext (falls vorhanden)
Sternchen-Bewertung
Tipp: Bewerten Sie das aktuelle Lied mit einem Klick an die richtige Stelle.

Anzeige von aktiven Tools (nicht im Bild)
Je nachdem welche Tools gerade aktiv sind, werden unter den Sternchen weitere Symbole eingeblendet.
Der Countdown zum Herunterfahren wurde aktiviert. Was genau wann passiert, erfahren Sie über den Hinweis, der erscheint, wenn Sie eine zeitlang den Mauszeiger auf diesem Symbol lassen.
Der Geburtstagsmodus wurde aktiviert. Nemp wird zu gegebener Zeit die Wiedergabe der Playlist für ein Geburtstagsständchen unterbrechen.
Der Webserver ist aktiviert um den Zugriff auf den Player über einen Webbrowser zu ermöglichen.
Scrobbeln ist aktiviert. Die Lieder, die Sie gerade hören, werden in Ihr Benutzerprofil auf last.fm übertragen.
Der Walkman-Modu ist aktiv, und der Ladezustand des Akkus ist schwach. Nemp fängt dann an zu leiern.

Zeitanzeige
Tipp: Schalten Sie per Klick um zwischen Zeit- und Restzeit-Anzeige.

Lautstärke
Über den senkrechten Schieberegeler können Sie die Lautstärke einstellen.

Fortschrittsanzeige
Anzeige und Änderung des Fortschritts im aktuellen Titel.

Steuerung des Players

Tipp: Einige Buttons haben mehrere Funktionen. Probieren Sie die rechte Maustaste!

Abspielreihenfolge



Die einzelnen Symbole bedeuten dabei
  • Alles wiederholen: Nemp läuft die Playlist von oben nach unten durch und startet dann von vorne.
  • Titel wiederholen: Der aktuelle Titel wird in einer Schleife wiederholt.
  • Zufallsmodus: Die Dateien werden in zufälliger Reihenfolge abgespielt. In den Optionen können Sie einstellen, wann ein bereits abgespielter Titel erneut abgespielt werden darf.
  • Kein wiederholen: Nemp spielt die Playlist ab und hält dann die Wiedergabe an.
Aufnahme

Der Aufnahme-Button ist nur bei Webradio-Sendern im mp3-Format aktiv.

Aufnahme nicht möglich, da gerade eine Datei abgespielt wird, oder das Format des Senders von Nemp nicht für die Aufnahme unterstützt wird.
Aufnahme möglich. Klicken Sie den Button um die Aufnahme zu starten.
Aufnahme läuft. Klicken Sie auf den Button, um die Aufnahme zu beenden.
Die Aufnahme wird in dem Verzeichnis gespeichert, das in den Einstellungen dafür angegeben wurde. Sie können das Verzeichnis über das Menü "Tools -> Verzeichnisse -> Aufnahmen" anzeigen lassen.
