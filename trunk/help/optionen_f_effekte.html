Effekte

Bei Änderung der Geschwindigkeit
Nemp kann die Wiedergabe eines Liedes schneller oder langsamer ablaufen lassen. Technisch sehr einfach lässt es sich realisieren, wenn man dabei eine Veränderung der Tonhöhe in Kauf nimmt. Das hört sich lustig an, ist aber nicht immer gewünscht.
Wenn man die CPU etwas mehr rechnen lässt, kann man diesen Effekt recht gut vermeiden.


Effekte/Equalizer
Geben Sie an, ob die Effekt- und Equalizer-Einstellungen gespeichert und beim nächsten Start von Nemp wieder hergestellt werden sollen.


Jingles
Geben Sie an, was passiert, wenn Sie einen Jingle abspielen. Damit man diesen ordentlich hört, ist es oft ratsam, die Lautstärke der Wiedergabe zu reduzieren.


Kopfhörer
Einfügemodus für Kopfhörer-Dateien steuert, was passiert, wenn Sie auf den "Einfügen-Button" in der Kopfhörer-Steuerung klicken. Zur Auswahl stehen dieselben Möglichkeiten wie beim Einfügen aus der Medienliste.
Außerdem können Sie einstellen, ob die Kopfhörer-Wiedergabe automatisch angehalten werden soll, wenn Sie die Kopfhörer-Steuerung ausblenden, d.h. auf einen anderen Tab wie Cover, Lyrics, Equalizer oder Effekte wechseln.
Wenn Nemp im Einzelfenster-Modus ist, wird die Kopfhörer-Wiedergabe immer angehalten, wenn Sie das Kopfhörer-Fenster ausblenden.
