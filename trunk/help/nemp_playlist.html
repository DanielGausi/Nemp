Die Playlist

Kernstück eines jeden Players ist die Playlist, also die Liste mit Liedern, die abgespielt werden sollen.


Dateien in die Playlist einfügen
Wählen Sie über das Hauptmenü "Playlist-> Dateien hinzufügen" und wählen Sie die gewünschten Dateien aus oder ziehen Sie per Drag&Drop einen Ordner oder einzelne Dateien in die Playlist. Die Playlist ist dabei der rechte Bereich - da, wo Playlist steht. Wenn Sie die Dateien woanders hinziehen, werden sie in die Medienbibliothek aufgenommen. Dazu später mehr.


Webradio
Sie können mit Nemp auch viele der gängigen Webradio-Sender abspielen. Wenn der Sender eine .pls-Datei anbietet, können Sie diese einfach mit Nemp öffnen.
Andernfalls können Sie den Sender auch direkt einfügen. Klicken Sie mit der rechten Maustaste auf den Play-Button und wählen Sie Webradio, oder sie gehen den Weg über die Webradio-Verwaltung.


Abspielreihenfolge
Die Reihenfolge, in der die Playlist abgearbeitet wird, können sie über den Button für den Wiedergabemodus beeinflussen:



Die einzelnen Symbole bedeuten dabei
  • Alles wiederholen: Nemp läuft die Playlist von oben nach unten durch und startet dann von vorne
  • Titel wiederholen: Der aktuelle Titel wird in einer Schleife wiederholt.
  • Zufallsmodus: Die Dateien werden in zufälliger Reihenfolge abgespielt. In den Optionen können Sie einstellen, wann ein bereits abgespielter Titel erneut abgespielt werden darf.
  • Kein wiederholen: Nemp spielt die Playlist ab und hält dann die Wiedergabe an.

Abspielreihenfolge modifizieren
Sie können die Dateien in der Playlist über das Menü sortieren lassen, oder einzelne Einträge per Drag&Drop innerhalb der Liste verschieben.


Vormerkliste
Sie können auch über die Zifferntasten 0..9 eine temporäre individuelle Reihenfolge festlegen. Markieren Sie einfach das Stück, dass Sie als nächstes hören wollen, und drücken sie die Taste '1'. Ein anderes markieren Sie mit '2' usw.
Wenn sie die Reihenfolge ändern wollen, könne n Sie das genauso tun.
Wenn Sie eine Markierung entfernen wollen, drücken Sie die '0'.


Playlisten Laden und Speichern
Nemp unterstützt aktuell Playlists in den Formaten .m3u, .m3u8, .pls sowie ein eigenes Format. Wiedergabelisten im .asx oder .wax-Format können nur gelesen werden.
