Nemp: Noch ein mp3-Player

Note: "Noch ein mp3-Player" translates to "Yet another mp3-player".

A mp3-Player for Windows
Current Version: 4.6

New in version 4.6

  • Support for iTunes-Tags
  • Faster start of playback (especially at files on network drives)
  • Modified settings dialog
  • Some minor changes at the GUI
  • Some minor bugfixes 

New in version 4.5

  • Support for Apev2-Tags
  • A-B-Repeat
  • Skip silence at the end of a track
  • Support vor metadata "disc number"
  • Settings: No splashscreen, switch to new enqueued track (when double clicking a file in the Windows Explorer and Nemp already running)
  • Loading of library file in the background (quicker start)
  • Some minor changes at the GUI
  • Some minor bugfixes

New in version 4.4 (extract)

  • New webserver with a lot of new features: Scrollbar for the current title, volume control, sorting Playlist, deleting items from the playlist, browsing the library by artist, album or genre, and an optional admin access with more rights than for normal users
  • The webserver can be customized using themes
  • New feature: Insert track from headphones into the playlist and start playback immediately from the currently selected position
  • Change: Drag&Drop into the playlist insert the files after the highlighted title, not before it

New in version 4.3 (extract)

  • Better support for CD-Audio
  • Feature "Cleanup library" extended (i.e. Selection dialog, whether missing files on missing drives should be deleted or not)
  • update of bass_fx.dll - needed for Windows8
  • Changing scrobbling to the new API

New in version 4.2 (extract)

  • Multimedia keys should now work properly (finally!)
  • Ctrl+V will insert files into the playlist at the highlighted position
  • Added a wizard for some main features
  • Everything, what happens in the background and changes audio files, create new ones (except configuration files), going online, ... is now OPT-IN: These settings are done by the wizard.

New in version 4.1 (extract)

  • Support for meta tags in ogg- and flac-files
  • Copy playlist including a proper playlist file by Ctrl+Shift+C
  • Feature "Copy playlist" to copy the playlist (e.g. to a USB drive) with renaming files (e.g. "01-artist-title.mp3")
  • Push-To-Talk through F8 (similar to jingles by F9)
  • Support for Logitech G15 display (other keyboards may be supported later, ask me for details)

New in version 4.0 (extract)

  • Better coverflow with automatic download for missing cover art
  • Browse mode "Tagcloud"
  • Better editing of audio files
  • Suppoert for Windows7 (e.g. taskbar buttons)
  • Party mode with enlarged display and reduced features
  • ...
