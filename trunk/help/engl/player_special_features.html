Special Features

Besides the common features, Nemp offers you some features you may not find on other players.

Jingles F9 Play a second title simultaneously. This playback holds as long as the key is pressed.
Push to Talk F8 Reduce the main volume as long as the key is pressed.
Extended
Copy&Paste		 Ctrl+Shift+C Copy the selected files into the clipboard (as done with Ctrl+C). In addition, Nemp will add a proper playlist file to the clipboard.
Copy playlist  (Menu) Copy all files from the playlist into a specified directory and (optionally) rename the files.
Prebook list 1,2,...,9,0 Insert the selected title into a prebooklist. Files in this list will be played next, regardless of the playback order.
Cue-Sheets   If an audio file comes with a matching *.cue file, Nemp will parse this file, list all tracks and offers you to jump directly to these tracks within the file.
Random playlist (Menü) Create a playlist with randomly selected titles from your playlist. You can specify some conditions (e.g. add only "jazz" or "songs from the 90ies")
CSV export (Menü) Export the media library in *.csv format
Shutdown (Menü) Nemp can stop itself or shutdown/suspend/hibernate your PC after some time.
If the shutdown timer is activated, a symbol will be shown in the main window.
