Extended display / controls


Cover-Art to the current track.
Tip: Nemp can download missing cover art automatically
Lyrics to the current track
Tip 1: You can set the lyrics manually in the properties dialog
Tip 2: Select some tracks in the title list of the media library and press Ctrl+L. Nemp will search the lyrics for the selected songs automatically (and quite often it will find them, too).
Equalizer
Tip: Right click on one of the controls will reset it to the default setting.
Effects
Tip: You can change in the settings dialog, how changing the speed will sound.

Besondere Effekte
Reverse playback
A-B-Repeat
Tip: If you selected the start point A and the end point B of the loop, you can click&drag the little controls in the main part of the player to change the loop.
Headphones
Tip: Install a second sound card on your PC. Very simple ones for a few Euros/Dollars/...  will work for that.

Special features
Copy the currently selected track of the playlist (or the title list of the media library) into headphone controls and starts playback.
Tip: You can use the shortcut Ctrl+H for that.
Copy the current track from the headphones control into the playlist.
Tip: There is a setting (Settings dialog -> Playlist) what exactly happens (enqueue at the end, after the current title, ...)
Copy the current track from the headphones control into the playlist and immediatly begins playback (through the main speakers) from the current position.
Tip: Use this feature, if you want to play a track with an annoying intro on a party. Scroll (in headphone mode) beyond the intro, wait until the current track (on main speaker) is near the end and click this button.
