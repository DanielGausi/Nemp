Maintenance of the Media Library

For a correct display and better search results it is necessary, that the files are tagged properly.

Nemp 4.6 supports the following types of metadata:
  • ID3-Tags (Version 1, 1.1, 2.2, 2.3 and 2.4) (used in *.mp3)
  • Ogg-Vorbis-Comments (used in *.ogg)
  • Flac-Metadata (used in *.flac)
  • Apev2-Tags (used in *.ape, *.mpc and some others)
  • iTunes Tags (used in *.m4a)
If you have many WindowsMediaAudio-files, you probably want to use the Windows Media Player. Nemp can read some data from *.wma files, but changing them is not possible.


Changing information
Tip 1: Click (slowly) two times on some information in the title list of the media library, or use F2 there.
Tip 2: Click on some information in teh detailed display next to the title list.


Detailed editing
If you want some more, you can use the "properties" dialog from the context menu (right click). There you can edit the ID3-tags directly, and change lyrics or cover art.
Tip: try the shortcut Ctrl+D (D as in Details)


Lyrics
Nemp can save lyrics to the songs, so you may search for a song which goes somehow like "in the town, where i was born". The lyrics may be typed in manually, or you can use the automatic search. This will use the database of http://lyrics.wikia.com/. Just select some files in the title list and choose "Get lyrics"


Additional tags
Nemp can save (almost) arbitrary many additional tags to the songs, which don't fit the scheme "Artist, title, Album, Genre". You can set these tags on your own, or use the most common set tags from the LastFM community. Detailed information to this topic here.


Refreshing data
After changing ID3 Tagswith other software, you may refresh the information in the media library.
Select "Refresh selected files" or "Refresh library" from the menu.


Deleting files
If you want to exclude some file from your media library,, select "Delete selected files from the library". The files are just deleted from the library, not from your harddrive.
