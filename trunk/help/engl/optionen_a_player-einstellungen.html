Configure the player according to your preferences. Some of the settings should be quie clear and are not mentioned here.

Output devices
If you use the "default" setting, Nemp will always use the default sound device on your system and switch instantly the playback, if you change the setting in the Windows Control Panel.

Fading
Nemp can fade between the tracks. Useful for many cases, but not for live recordings or classical music.
Tip: For a fading between two tracks without any silence, you should use "Silence detection" with a treshold of -20dB (or even -10dB) and a fading time of 4000 ms on title change.

Playlist

The Default action for audio files is used when you hit the enter key (or double click) after selecting a title in the title list of the media library to add it into the playlist. 

The Default setting for headset is used when you click the button to add the current headphone file into the playlist.

Remember track position when playing a song directly from the library is used when you choose the "Just play" option above. When the "just played" track is completely played (or aborted by the user), Nemp will continue at the exact position of the previously played track.
Tip: Use this to preview a track without changing the the playlist itself.

Webstreams

Parse stream playlist 
Many webradio stations offer you a *.pls file to tune in. In fact, this is a well known format to store playlists (and supported by Nemp). Quite often these pls files contains a list of several servers. Sometimes all servers will stream the same content, but maybe with a different quality setting or encoding.
Nemp can parse the pls file and add all the servers to the playlist, or Nemp can just add the pls file itself to the playlist, and the server is choosen automatically.


Recording settings
Geben Sie an, in welchem Verzeichnis Nemp die Aufnahmen von Webradio speichern soll, und welche Dateibenennung sie bevorzugen.
Der Punkt "Bei Titelwechsel neue Datei beginnen" funktioniert ledier nur dann richtig, wenn der Sender vernünftige Tags mitsendet, die einen neuen Titel ankündigen. Anhand der reinen Audiodaten kann leider kein Titelwechsel erkannt werden.
Sie können die Dateien auch nach Größe oder Dauer aufteilen. Beachten Sie dabei, dass das nur ungefähre Werte sind. Die Dauer kann um einige Sekunden und die Größe um einige Kilobyte von der gewünschhten Größe abweichen.
Hinweis: Nemp kann nur Sender aufnehmen, die im mp3- oder im aac-Format senden. Sender, die im Windows-Format oder Ogg-Vorbis senden, können nicht aufgenommen werden.

Effekte

Änderung der Geschwindigkeit
Nemp kann die Wiedergabe eines Liedes schneller oder langsamer ablaufen lassen. Technisch sehr einfach lässt es sich realisieren, wenn man dabei eine Veränderung der Tonhöhe in Kauf nimmt. Das hört sich lustig an, ist aber nicht immer gewünscht.
Wenn man die CPU etwas mehr rechnen lässt, kann man diesen Effekt recht gut vermeiden.
