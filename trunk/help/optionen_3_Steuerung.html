Steuerung

Multimediatasten
Leider ein ganz besonderes Thema.  Diese Tasten verhalten sich von System zu System sehr unterschiedlich. Auf 32-Bit-Systemen kann Nemp versuchen, die Multimediatasten global abzufangen um so auch dann auf diese Eingaben zu reagieren, wenn nemp nicht den Eingabefokus hat.
Auf Systemen mit Logitech iTouch darf dieses abfangen aber nicht passieren - das führt zu sehr seltsamen Effekten.
Über die Konfiguration kann Nemp testen, ob iTouch oder ein vergleichbares System installiert ist.

Warnung: Auf 64-Bit-Systemen kann das zwar auch versucht werden, wird aber nicht empfohlen.


Hotkey-Konfiguration
Über globale Hotkeys kann Nemp bedient werden, wenn es nicht den Eingabefokus hat. Sie sollten hier kombinationen wählen, die nicht von Ihren anderen Programmen für andere zwecke benutzt werden.


Tab-Taste
Über die Tabulator-Taste kann Nemp auch fast ohne die Maus gesteuert werden. Sie können hier einschränken, ob einige Kontrollelemente über die Tab-Taste markiert werden kann. Andernfalls können Sie diese Buttons nur mit Hilfe der Maus bedienen.
