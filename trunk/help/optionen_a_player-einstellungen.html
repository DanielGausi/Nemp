Player-Einstellungen

Ausgabegeräte
Steuert, auf welchem Gerät die Hauptausgabe und die Kopfhörerausgabe stattfindet. Wenn Sie zwei Soundkarten in Ihrem Rechner eingebaut haben, können Sie dadurch z.B. auf einem Kopfhörer an der einen Soundkarte etwas anderes hören als auf der Hauptausgabe.


Fading
Steuert, ob und wie zwischen einzelnen Stücken übergeblendet wird.
  • Ein Fading zwischen zwei Titeln sorgt für einen sanften Übergang von einem Titel zum nächsten
  • Ein Fading bei Positionswechsel sorgt für ein nicht ganz so abruptes springen innerhalb eines Titels, wenn Sie über den Schieberegeler im Hauptfenster an eine andere Position im Titel springen.
Das Fading kann bei kurzen Stücken ignoriert werden, oder wenn Sie den Stopp- oder Pause-Button anklicken.
