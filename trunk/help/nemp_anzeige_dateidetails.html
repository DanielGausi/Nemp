Eigenschaften, Dateidetails

Wenn Sie eine Datei in der Playlist oder in der Medienbibliothek markieren, und über das Popupmenü den Punkt "Eigenschaften" wählen, dann bekommen Sie in einem weiteren Fenster eine ganze Reihe Eigenschaften dieser Datei angezeigt.

Allgemein
Auf der ersten Seite finden Sie die Daten, die in der Medienbibliothek über diese Datei gespeichert sind. Über den Button "Explorer" bekommen Sie die Datei im Windows-Explorer angezeigt, über "Eigenschaften" erhalten Sie den Windows-Eigenschaftsdialog dazu.
Der Button "Aktualisieren" liest die Daten erneut aus dem ID3-Tag (bei mp3, sonst halt andere Metatags) und überschreibt die Daten aus der Medienbibliothek mit diesen Daten.
Außerdem finden Sie auf dieser Seite die Bilddateien, die Nemp für die Coverauswahl gefunden hat. Durch einen Doppelklick auf das Bild wird es mit dem Standard-Bildbetrachter geöffnet.

Id3-Tags (mp3) / Vorbis Kommentare (ogg, flac)
Auf der zweiten Seite können Sie die Metadaten der Datei bearbeiten. Unterstützt werden dabei mp3-Dateien (ID3-Tags), sowie ogg- und flac-Dateien (Vorbis Kommentare).
Wenn Sie hier Daten verändern, und unten auf "Übernehmen" klicken, werden die Daten in die Datei geschrieben. Dies ist unabhängig von der Einstellung bei den Metadaten.
Da die Bewertungen auch unabhängig von der Medienbibliothek gespeichert werden können, gibt es hier einen (oder zwei) Button(s). Über "Zurücksetzen" setzen Sie die Bewertung und den Abspielzähler zurück auf Null.
Wenn die Bewertung aus der Medienbibliothek nicht mit der in den Tags gespeicherten Bewertung übereinstimmt, gibt es auch einen Button "Bewertung synchronisieren", der die Daten aus der Medienbibliothek in den Tag übernimmt.

Lyrics & Bilder
Auch Liedtexte und Cover können direkt in der Musikdatei gespeichert werden (Cover nicht bei *.ogg). Die Liedtexte können auch automatisiert über lyrics.wikia.com abgerufen werden.

MP3- und ID3-Details
Hier werden weitere Informationen über die mp3-Datei angezeigt. Viele Daten sind für den Normalanwender relativ unwichtig - einzig die ID3v2-Informationen sind häufiger interessant: Wenn die Version 2.4.0 verwendet wird, hat Windows damit einige Probleme. Wenn es unter Windows zu Anzeigeproblemen kommt, dann liegt das häufig an dieser Tag-Version.

Tags für die Tagwolke
Nemp kann auch weitere Tags in den Dateien speichern, die von anderen Playern nicht verwendet werden können. Diese "weiteren Tags" werden für die Bildung der Tagwolke benutzt. Diese erweiterten Tags können völlig beliebig gewählt werden. Es sind sowohl alternative Genre-Einordnungen möglich, aber auch sehr persönliche wie "Lieblingslied" oder "Skiurlaub 2010/11".
