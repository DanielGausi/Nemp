Allgemeine Einstellungen

Beim Starten von Nemp
  • Beim Start mit der Wiedergabe beginnen.
  • Letzte Position im Track speichern.  Wiedergabe genau dort fortführen, wo sie beim letzten Mal unterbrochen wurde. besonders sinnvoll bei langen Titeln, z.B. Hörbüchern.
  • Falls verfügbar: Wiedergabe mit neuer Datei beginnen. Spielt beim Start die Datei ab, die per Doppelklick im Explorer abgespielt wird und die das Starten von Nemp bewirkt hat.
  • Lade/Speichere Medienbibliothek beim Start. Ist das Häkchen hier nicht gesetzt, wird die Medienbibliothek beim Start nicht geladen und am Ende nicht gespeichert.
  • Mehrere Instanzen erlauben. Normalerweise sollte Nemp nur einmal gestartet werden, nicht mehrmals.
  • Minimiert starten. zeigt beim Start nicht das Programmfenster, sondern minimiert es direkt.
  • Im Vordergrund halten. Behält im Einzelfenster-Modus den Playerteil im Vordergrund
