Webserver

Konfigurieren Sie den Nemp Webserver, um ihren Freunden im lokalen Netz oder auch vom anderen Ende der Welt Zugriff auf ihre Medienbibliothek zu geben.

Mehr zum Webserver hier.

Konfiguration
  • Über das Theme kann das Aussehen und der Funktionsumfang festgelegt werden. Zum Anlegen weiterer Themes sei auf die Dokumentation im Ordner <Nemp>\HTML\ verwiesen
  • Der Port ist standardmäßig 80 (http), kann aber geändert werden, wenn z.B. Parallel ein "echter" Webserver (z.B. mit xampp) betrieben wird.
  • Der Zugriff  kann auf das lokale Netz beschränkt werden
  • Der Webserver kann auch direkt mit Nemp gestartet werden.
  • Als Administrator hat man ggf. weitere Zugriffsmöglichkeiten (aktuell nur im Theme "Party")
User Berechtigungen
Auch wenn über das Theme eingestellt werden kann, welche Funktionen im Browser auftauchen, so sind diese dann nicht blockiert, wenn z.B. ein erfahrerener Nemp-Nutzer die betreffenden Kommandos direkt in die Adresszeile eintippt.
Das kann durch das Setzen (bzw. Entfernen) der User-Berechtigungen verhindert werden.
Das bewirkt dann, dass erstens die entsprechenden Links nicht im Browser auftauchen, auch wenn sie im Theme vorgesehen sind und zweitens, dass manuelle Kommandos ignoriert werden.

Eigene IP-Adressen
Auflistung der lokalen IP-Adresse(n). Über diese ist der Nemp-Webserver im lokalen Netz erreichbar. Für Zugriff von außerhalb ist die "Internet-IP" erforderlich, die per Klick auf den Button ermittelt werden kann (das läuft über ein PHP-Script auf www.gausi.de)
