Zufalls-Playlisten erstellen

Bei einer umfangreichen Musiksammlung ist es manchmal schwierig, eine Auswahl zu treffen.

Nemp kann daher aus den Titeln in der Medienbibliothek eine zufällige Playlist erstellen. Dem Zufall können Sie dabei etwas auf die Sprünge helfen. Um eine zufällige Playlist zu erstellen wählen sie über das Menü "Playlist -> Zufallsplaylist erstellen".


Allgemeine Einstellungen
Geben Sie an, ob die zufälligen Titel aus der gesamten Medienbibliothek gezogen werden sollen, oder nur aus der aktuell angezeigten Liste.
Die maximale Anzahl gibt die Länge der Playlist an, die Sie erstellen wollen.


Zeitraum
Geben Sie an, ob Sie nur Dateien aus bestimmten Jahren zulassen wollen. Maßgeblich hierfür ist das Jahr, das in den ID3-Tags der Audiodateien angegeben ist, nicht das Erstelldatum der Datei.


Dauer
Schließen Sie sehr kurze oder zu lange Titel aus der Playlist aus


Bewertung
Lassen Sie nur besonders hoch oder besonders schlecht bewertete Titel in der zufälligen Playlist zu.


Tags
Schränken Sie die Wahl der Titel ein, indem Sie bestimmte Tags fordern. Sie können dabei mehrere Tags ankreuzen und angeben, wieviele der angegebenen Tags tatsächlich in den Dateien für die Playlist gesetzt sein müssen.
Angezeigt werden in der Liste die 10-500 häufigsten Tags (je nach Auswahl). Die Zahl in Klammern gibt an, wie oft der jeweilige Tag in der Medienbibliothek vorkommt.


Einfügemodus
Im unteren Teil des Zufalls-Playlist-Fensters können Sie noch angeben, ob Sie die zufälligen Dateien in die bestehende  Playlist einfügen möchten, oder eine neue Playlist erstellen möchten.
