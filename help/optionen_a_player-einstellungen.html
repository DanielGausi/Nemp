Player-Einstellungen

Stellen Sie den Player nach ihren Wünschen ein.

Tipp: Für einen Übergang zwischen zwei Titeln in der Playlist ohne Stille aktivieren sie "Stille erkennen" und "Fading". Je näher der Schwellwert an 0dB liegt, desto früher wird der neue Titel gestartet.

Playlist

Standardaktion für Audiodateien wirkt sich auf das Einfügen per Doppelklick oder <Enter> von der Titelanzeige der Medienbibliothek in die Playlist aus.

Standardaktion für Kopfhörer wirkt sich auf das Einfügen vom Kopfhörer-Player in die Playlist aus, wenn Sie auf den Button klicken.

Merke Position im aktuellen Track beim Abspielen direkt aus der Medienbibliothek bewirkt, dass bei "Datei einfach abspielen" als Standardaktion und Klick auf "nächster Titel" die Wiedergabe der Playlist genau dort fortgeführt wird, wo sie vorher unterbrochen wurde.
Tipp: Damit ist ein schnelles Reinhören in einen Titel möglich, ohne die eigentliche Playlist zu verändern.

Webradio

Stream-Playlist parsen
Viele Sender bieten eine *.pls-Datei zum Anhören an. Das ist eigentlich ein Playlisten-Format, in dem mehrere Dateien oder Adressen angegeben sein können. Bei Webradio-Sendern werden oft mehrere Adressen angegeben, die alle dasselbe senden, aber die Last auf mehrere Server verteilen.
Nemp bietet die Möglichkeit, all diese Adressen in die Playlist aufzunehmen - dann erscheint ein Sender ggf. mehrfach in der Playlist. Ansonsten wählt Nemp intern eine zufällige Adresse aus dieser Liste aus.

Aufnahmeoptionen
Geben Sie an, in welchem Verzeichnis Nemp die Aufnahmen von Webradio speichern soll, und welche Dateibenennung sie bevorzugen.
Der Punkt "Bei Titelwechsel neue Datei beginnen" funktioniert ledier nur dann richtig, wenn der Sender vernünftige Tags mitsendet, die einen neuen Titel ankündigen. Anhand der reinen Audiodaten kann leider kein Titelwechsel erkannt werden.
Sie können die Dateien auch nach Größe oder Dauer aufteilen. Beachten Sie dabei, dass das nur ungefähre Werte sind. Die Dauer kann um einige Sekunden und die Größe um einige Kilobyte von der gewünschhten Größe abweichen.
Hinweis: Nemp kann nur Sender aufnehmen, die im mp3- oder im aac-Format senden. Sender, die im Windows-Format oder Ogg-Vorbis senden, können nicht aufgenommen werden.

Effekte

Änderung der Geschwindigkeit
Nemp kann die Wiedergabe eines Liedes schneller oder langsamer ablaufen lassen. Technisch sehr einfach lässt es sich realisieren, wenn man dabei eine Veränderung der Tonhöhe in Kauf nimmt. Das hört sich lustig an, ist aber nicht immer gewünscht.
Wenn man die CPU etwas mehr rechnen lässt, kann man diesen Effekt recht gut vermeiden.

Jingles, Push-To-Talk
Stellen Sie ein, wie stark Nemp die Lautstärke der Hauptwiedergabe herabsetzen soll, wenn parallel Jingles abgespielt werden (F9) oder die Push-To-Talk-Taste (F8) gedrückt wird.

Walkman-Modus
Wenn Sie Nemp auf einem Rechner laufen lassen, der nur per Akku mit Strom versorgt wird, dann kann Nemp den Ladezustand des Akkus überwachen. Sinkt dieser unter 10%, dann beginnt Nemp, die Wiedergabe zu verzerren - ähnlich wie es bei den tragbaren Kassetten-Abspielgeräten die Regel war.  Einfach eine weitere spaßig-nerdige Einstellung ohne tieferen Sinn.
