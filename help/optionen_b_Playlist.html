Playlist

Standardaktion für Audiodateien
Regelt das Verhalten, wenn Sie in der Medienliste einen Doppelklick auf ein Lied ausführen. Sie können dabei
  • das Stück am Ende der Playlist einfügen
  • die aktuelle Playlist löschen und mit dem markierten Stück eine neue playlist beginnen
  • das Stück  hinter den aktuellen (bzw. ans Ende der Vormerk-Liste) einfügen
  • das Stück einfach abspielen, ohne die Playlist zu verändern. Die Wiedergabe der Playlist wird dann unterbrochen und nach Ende des markierten Stückes fortgesetzt.

Zufallswiedergabe
Auch wenn es einem intuitiv so gar nicht klar ist: Bei einer wirklich zufälligen Wiedergabe kommt es recht häufig dazu, dass ein Lied zweimal innerhalb kurzer Zeit abgespielt wird. Nemp kann das bis zu einem gewissen Grad vermeiden.
Mit dem Schieberegler können Sie den Anteil der Playlist einstellen, der mindestens abgespielt werden muss, bevor ein Titel wiederholt werden darf.


Verhalten der Playlist
  • Dateien beim Start untersuchen bedeutet, dass die Audiodateien beim laden aus der Playlist untersucht werden, und sich nicht darauf verlassen wird, dass die Daten für z.B. Dauer aus der Playlist-Datei korrekt sind.
  • Vollständig abgespielte Dateien können automatisch aus der Playlist entfernt werden. Sie können das automatische Löschen unterbinden, wenn Sie während der Wiedergabe im Titel spulen, zwischen durch pausieren, stoppen oder manuell zum nächsten Titel springen.
  • Nemp kann die Playlist automatisch alle 5 Mniuten sichern. Sollte Nemp doch einmal abstürzen, wird beim nächsten Start automatisch diese temporäre Playlist geladen.
  • Nach dem letzten Stück neu mischen bewirkt ein mischen der Playlist, wenn sie einmal durchgelaufen ist. Das bewirkt dann eine Art zufällige Wiedergabe, auch wenn der Wiedergabemodus auf "Alles Wiederholen" steht.
  • Wenn Sie auf "Nächster Titel" klicken, kann Nemp entweder zum nächsten Eintrag in der Playlist springen, oder nur zum nächsten Eintrag im Cuesheet zur aktuellen Datei. Mehr dazu hier.
  • Wenn Sie einen Titel direkt aus der Medienbibliothek abspielen (s.o., letzter Punkt), können Sie einstellen, ob die Playlist das unterbrochene Stück von Anfang an wiedergeben soll, oder an der unterbrochenen Stelle starten soll.
