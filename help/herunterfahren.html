Herunterfahren

Nemp kann sich nach einer gewissen Zeit selbst beenden und den Computer dabei herunterfahren.

Sie haben dabei zur Auswahl
  • Nemp anhalten 
  • Nemp beenden
  • Windows in den Energiesparmodus versetzen
  • Windows in den Ruhezustand versetzen
  • Windows beenden
Für jede dieser Aktionen können Sie einen Zeitpunkt angeben, oder auf das Ende der Playlist legen.

Wenn der Timer zum Herunterfahren aktiv ist, wird im Hauptfenster ein Symbol  eingeblendet.
