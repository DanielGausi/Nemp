Verwaltung von Webradio

Nemp kann nicht nur Dateien verwalten, sondern auch eine Liste von Webradio-Sendern, die Sie regelmäßig hören.

Webradio - Favoriten
In der klassischen Browse-Ansicht werden neben den Dateien auch die bevorzugten Webradio-Sender angezeigt. Sie können diese Liste über das Menü "Medienbibliothek -> Webradio-Sender verwalten" bearbeiten.
Unter "Favoriten" können Sie neue Sender hinzufügen. Den Namen können Sie frei wählen, als URL sollten Sie die Adresse des Senders eingeben. Sehr häufig bekommt man diese auf der jeweiligen Webseite - in der Regel findet sich dort ein Link, der auf *.pls endet. Diesen Link müssen Sie im URL-Feld eintragen.

Hinweis: Viele Sender bieten unterschiedliche Links für den Windows Media Player, Winamp oder den real Player an. Am besten sollte Nemp mit dem Winamp-Link klarkommen.

Sie können die Favoriten Liste auch Ihren Wünschen entsprechend sortieren.


www.shoutcast.com
Auf www.shoutcast.com gibt es eine fast unzählige Auswahl von Radiostationen. Auf dieser Webseite gibt es auch eine Suchfunktion, die Sie auch direkt von Nemp aus benutzen können.
Suchen Sie einfach nach einem Sendernamen, oder einen Interpreten, den Sie hören möchten und wählen Sie einen Sender aus der Trefferliste aus. Beachten Sie bitte, dass der angezeigte aktuelle Titel nicht immer stimmt. Die Pflege dieser Information liegt in der Verantwortung des Senders, und nicht alle liefern dort minuten-aktuelle Daten.

Sie können einen Sender per Doppelklick in die Playlist einfügen, oder ihre Favoritenliste ergänzen.


Stream-Playlist parsen
Viele Sender bieten eine *.pls-Datei zum Anhören an. Das ist eigentlich ein Playlisten-Format, in dem mehrere Dateien oder Adressen angegeben sein können. Bei Webradio-Sendern werden oft mehrere Adressen angegeben, die alle dasselbe senden, aber die Last auf mehrere Server verteilen.
Nemp bietet die Möglichkeit, all diese Adressen in die Playlist aufzunehmen - dann erscheint ein Sender ggf. mehrfach in der Playlist. Ansonsten wählt Nemp intern eine zufällige Adresse aus dieser Liste aus.
