Troubleshooting

Playback does not work - I don't hear anything
  • Check the volume settings
  • Connect all cables. Do other software have sound?
  • Try another output device (Preferences -> Player settings)
  • After hibernating teh system, teh player may need to be reinitialized: Preferences -> Hibernate/Stand by
Nemp doesn't start
  • On some systems the 3D-Coverflow may cause some issues.
  • Use "NempSafemode.exe" for starting Nemp. This will activate the old 2D-Coverflow
Playback stutters
  • Enlarge the buffer size: Preferences -> Player (extended)
Playback is distorted
  • Check the Equalizer settings.
  • Deactivate Floating Point Channels or use Software Mixing: Preferences -> Player (extended)
    ("Einstellungen -> Erweiterte Einstellungen")
Multimedia keys don't work
  • Check settings Preferences -> Controls (Media keys)
The Coverflow shows only the default cover, but the player shows the correct ones
  • For a better performance, Nemp uses only cached images for the Coverflow, which are created when searching for audiofiles. They are stored in the directory you can reach by Tools->Directories->Data.There should be a subdirectory with a lot of files like "BE321C7AE594416632C0C6024641B1D1.jpg". Nemp needs write access to this directory.
The Coverflow doesn't find any cover art
  • Check Preferences -> File management -> Cover and activate the online cover search
  • Check your internet connection, firewall and so on
  • The songs need to be tagged properly for an automatic cover search.
  • Cover search doesn't work well on sampler CDs, as the last.fm database is not very consistent in these cases. Sometimes an album is called "Bravo Hits 60", and the next one "Bravo Hits Vol. 61"...
The Coverflow is only 2D, not 3D
  • Probably you have started the NempSafemode.exe once. This changes the Coverflow permanently.
  • Uncheck Preferences -> Viewing Options -> Coverflow -> use classic Coverflow 
  • Nemp needs for the 3D-Cowerflowthe files OpenGL32.dll und GLU32.dll, which should be located in your system directory.
  • Try to update the driver of your video card.
I use Windows 7, but I dont see any buttons in the Taksbar
  • use a setting without a (*) at Preferences -> General settings -> taskbar and tray
The Deskband doesn't work
  • After installing the Deskband, you need to log off once to activate the deskband.
Webradio doesn't work: "Unknown format"
  • try to get another URL for the Webstream. If available, the Link for "Winamp" should work with Nemp.
  • Activate "Parse Stream-Playlist" at Preferences -> Player settings -> Webstreams
I can't record Webstreams
  • Recording of webstreams works only for mp3- and aac encoded streams.
When recording webstreams, I get only one huge file
  • Activate setting "save different titles in seperate files"
  • This only works, if the the stream contains metadata for the title. Many stations add proper metadata, some don't.
Scrobble doesn't work
  • You need an account on last.fm
  • Check your internet connection and firewall settings
