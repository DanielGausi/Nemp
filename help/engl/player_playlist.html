The playlist

The essence of every player is the playlist - the list of the songs that will be played next.


Add files to the playlist
Tip 1: Try Drag&Drop
Tip 2: Try Copy&Paste


Webradio
With Nemp you can listen to many webradio stations. If a station offers a .pls file for download, you can open this file with Nemp, and Nemp will start the playback.
Tip 1: Register Nemp as the default player for playlist files to load these files directly from the webbrowser.
Tip 2: Organize your favorite stations through "Media library -> Manage webradio stations" (Shortcut Ctrl+W).


Playback order
Switch between fouir different modes:


  • Repeat everything. Nemp will play every title in the playlist and repeat after the last one
  • Repeat current track: Only the current track is repeated in a loop
  • Random mode: The tracks in the playlist a played randomly. Note: In the extended playlist settings you can adjust between "really random" (which means that a song can be played twice within a short time) or "Avoid repetitions" (which means if a song was played, it is blocked for some time).
  • Repeat off: Nemp will play every title in the playlist and stop after the last one.

Modify playback order
Tip 1: Nemp can sort the playlist
Tip 2: Try Drag&Drop within the playlist
Prebook list
With the numeric keys (0..9) you can set an individual playback order. Just select a title in the playlist and hit "1". Nemp will then play this title next, regardless of the playback order. To remove a title from the prebook list press "0".
