Fehlerbehebung

Die Wiedergabe funktioniert nicht - ich höre nichts.
  • Stellen Sie sicher, dass die Lautstärke nicht zu niedrig ist
  • Stellen Sie sicher, dass alle Kabel verbunden sind. Funktioniert die Soundausgabe mit anderen Programmen?
  • Versuchen Sie, die Wiedergabe auf eine andere Soundkarte zu legen:
    "Einstellungen -> Player-Einstellungen"
  • Wenn Sie den Computer aus dem Ruhezustand aufwecken: Versuchen Sie, den Player neu zu initialisieren: "Einstellungen -> Erweiterte Einstellungen"
Nemp startet nicht - der bleibt beim Start irgendwie hängen
  • Auf einigen wenigen Systemen mit älterer Hardware kann es zu Problemen mit dem neuen Coverflow kommen, der gewisse 3D-Features benötigt. Alte Grafikkarten unterstützen das eventuell nicht.
  • Starten Sie Nemp über die Datei "NempSafemode.exe". Dann wird eine Einstellung dauerhaft geändert, so dass Nemp immer auf den alten 2D-Coverflow zurückgreift.
Die Wiedergabe stottert
  • Vergrößern Sie die Buffergröße unter "Einstellungen -> Erweiterte Einstellungen"
Die Wiedergabe ist verzerrt
  • Ist der Equalizer aktiv? Extreme Einstellungen erzeugen fiese Nebeneffekte.
  • Versuchen Sie, die Floating-point Channels zu deaktivieren oder das Mixing auf "Software" umzustellen
    ("Einstellungen -> Erweiterte Einstellungen")
Die Wiedergabe läuft irgendwie doppelt
  • Überprüfen Sie, ob Effekte wie Echo oder Hall aktiviert sind und setzen Sie diese ggf. zurück.
  • Überprüfen Sie, ob die Wiedergabe im Kopfhörer läuft. Wenn auf Ihrem System nur eine Soundkarte installiert ist (dürfte in den meisten Fällen so sein), dann laufen Haupt- und Kopfhörerausgabe auf derselben Karte.
    Wenn Sie mehr als eine Soundkarte in Ihrem Rechner haben, können Sie die Ausgabe auf die beiden Karten aufteilen.
  • Überprüfen Sie, ob Nemp in mehreren Instanzen läuft. Normalerweise sollte immer nur eine Instanz von Nemp gestartet werden können, was man in den Einstellungen aber ändern kann.
Die Multimediatasten auf meiner Tastatur funktionieren nicht
  • Aktivieren Sie in den Einstellungen -> Steuerung die Option "Multimediatasten verwenden"
Im Coverflow erscheint nur das Standard-Cover, obwohl der Player selbst das richtige Cover anzeigt
  • Für die Anzeige im Coverflow benutzt Nemp aus Performancegründen nur die kleinen Coverbilder, die beim Aufbau der Medienbibliothek generiert werden. Wenn diese nicht vorhanden sind, weil sie gelöscht wurden oder Nemp sie nicht anlegen konnte (z.B. beim Start von CD/DVD), dann wird im Coverflow nur das graue Standard-Cover angezeigt.
    Sie erreichen das Cover-Verzeichnis über "Tools -> Verzeichnisse -> Daten". In diesem Verzeichnis sollte ein Unterordner "Cover" vorhanden sein mit lauter kleinen Dateien wie "BE321C7AE594416632C0C6024641B1D1.jpg".
  • Kopieren Sie Nemp auf Ihre Festplatte oder einen USB-Stick, auf dem Sie Schreibzugriff erlauben.
Der Coverflow findet keine weiteren Cover
  • Ist die Suche nach Covern im Internet aktiviert?
    "Einstellungen -> Player und Cover -> Fehlende Cover von last.fm herunterladen",
  • Ist eine Internetverbindung vorhanden?
  • Blockiert eine Firewall die Kommunikation von Nemp? Setzen Sie in Ihrer Firewall eine entsprechende Erlaubnis.
  • Sind die Lieder "vernünftig" benannt? Ein Cover kann nur dann gefunden werden, wenn der Album-Interpret und -Titel richtig ermittelt werden kann
  • Bei Samplern funktioniert die Coversuche generell nicht richtig, da hier leider die Datenbank, auf die Nemp zurückgreift, nicht ganz konsistent ist. Da ist dann ein Cover z.B. unter "Bravo Hits 60" abgelegt, und das nächste unter "Bravo Hits Vol. 61".
Der Coverflow sieht nicht so aus wie auf den Screenshots, sondern flach und unschön!
  • Vermutlich haben Sie Nemp einmal im Safemode gestartet. Dabei wird dauerhaft eine Einstellung geändert, um Probleme auf älterer Hardware zu umgehen.
  • Deaktivieren Sie unter "Erweiterte Einstellungen" die Option "Alten Coverflow verwenden". Wenn Nemp daraufhin nicht mehr reagiert oder nicht mehr neu startet, dann starten Sie Nemp einmalig über die Datei "NempSafemode.exe" - dadurch wird der Coverflow wieder auf die alte Version zurückgesetzt.
  • Nemp benötigt für den neuen Coverflow die Dateien OpenGL32.dll und GLU32.dll, die normalerweise im Systemverzeichnis (c:\Windows\System32) zu finden sein sollten. Auf einigen wenigen Systemen fehlen diese Dateien.
  • Sie können versuchen, diese Dateien nachträglich zu installieren oder zu aktualisieren, indem Sie z.B. die Treiber für Ihre Grafikkarte aktualisieren.
  • Es kann sein, dass Ihre Hardware die nötigen 3D-Funktionen nicht unterstützt. In diesem Fall hilft nur der Umstieg auf einen neueren Rechner oder eben der alte Coverflow.
Ich habe Windows 7, aber in der Fenster-Vorschau sind keine Buttons!
  • Stellen sie unter "Einstellungen -> Taskleiste und Tray" einen Punkt ein, der nicht mit einem (*) markiert ist
Das Deskband funktioniert nicht
  • Nach der Installation müssen Sie sich kurz abmelden und neu anmelden, damit die Änderungen in Windows wirksam werden.
  • Sie müssen dann das Deskband aktivieren. Über einen Rechtsklick auf die Taskleiste wählen Sie unter "Symbolleisten" das Nemp-Deskband aus
Beim Abspielen von Webradio kommt "Unbekanntes Format"
  • Versuchen Sie, auf der Webseite eine alternative URL für den Webstream zu finden. Wenn verfügbar, sollte der Link für "Winamp" mit Nemp gut funktionieren.
  • Aktivieren Sie die Option "Parse Stream-Playlist" (in der Stream-Verwaltung oder unter "Einstellungen -> Webradio") und probieren Sie einen anderen Links aus
Ich kann kein Webradio aufnehmen
  • Nemp kann nur mp3- und aac-Streams aufnehmen. Ogg und WMA geht nicht.
Bei der Aufnahme sind alle Titel zusammen, obwohl ich die Option "Neue Datei bei neuem Titel" aktiviert habe.
  • Das Trennen nach Titeln funktioniert nur, wenn der Sender passende Titel-Informationen mitsendet. Das machen leider nicht alle Sender.
Der Webserver funktioniert nur innerhalb des lokalen Netzwerkes
  • Ist der Zugriff von außerhalb erlaubt?
  • Für den Zugriff von außen müssen sie unter Umständen ihren Router so konfigurieren, dass er den Port 80 (TCP) auf die lokale Adresse Ihres Rechners weiterleitet.
    Wie das genau geht, entnehmen Sie bitte der Dokumentation Ihres Routers. Hilfreich sind dabei oft auch Seiten rund ums Thema Filesharing - da müsssen auch diverse Ports weitergeleitet werden.
Das Scrobbeln funktioniert nicht
  • Stellen Sie sicher, dass Sie einen gültigen Account bei last.fm haben
  • Stellen Sie sicher, dass eine Internetverbindung besteht und keine Firewall die Verbindung blockiert
  • Sie müssen Nemp Zugriff auf Ihr Benutzerprofil auf last.fm erlauben - Details dazu hier.
Oben im Menü erscheinen ständig "Nachrichten"
  • Diese Meldungen informieren Sie darüber, wenn etwas nicht stimmt, aber dieses "etwas" nicht so schlimm ist, dass Sie durch ein Hinweisfenster genervt werden müssen. Solche Fehler können beim Scrobbeln auftreten, beim Einlesen von Metadaten, oder beim Speichern von Metadaten.
  • Eine solche Nachricht wird auch dann generiert, wenn durch eine Verweigerung des Schnellzugriffs auf die Metadaten eine Speicherung der Bewertung verhindert wurde. Da dadurch die Daten in der Medienbibliothek inkonsistent zu denen in den ID3-Tags werden, wird ein entsprechender Hinweis erzeugt. 
  • Nemp ist für mp3, ogg, und flac-Dateien optimiert andere Dateitypen werden in der medienbibliothek nur sehr eingeschränkt unterstützt.
  • Deaktivieren Sie ggf. die automatische Bewertung der Dateien, oder aktivieren Sie den Schnellzugriff auf die Metadaten. Dann sollten die Nachrichten deutlich weniger werden.
