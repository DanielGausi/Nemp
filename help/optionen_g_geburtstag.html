Geburtstagsmodus

Konfigurieren Sie den Geburtstagsmodus von Nemp.

Sie können für das Geburtstagslied und das Countdownlied per Klick das gerade markierte Lied im Hauptfenster übernehmen. Es spielt dabei keine Rolle, ob Sie ein Lied in der Playlist oder der Medienbibliothek markiert haben.

Hinweis: Wenn Sie ein Lied für einen Countdown verwenden, dann wird dieser Countdown vor dem Event gestartet, so dass das eigentliche Lied für das Event zur angegebenen Zeit startet.
