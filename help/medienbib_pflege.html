Pflege der Medienbibliothek

Für eine korrekte Sortierung und ein schnelles Finden der Dateien ist es wichtig, dass die einzelnen Audiodateien mit ordentlichen ID3-Tags versehen sind.

Hinweis: Aktuell kann Nemp nur ID3-Tags in mp3-Dateien verändern. Andere Formate werden nicht unterstützt (oder nur lesend).


Informationen bearbeiten
Sie können die Information direkt im Hauptfenster ändern.
  • In der Tabellen-Ansicht die entsprechende Zelle markieren, und zweimal langsam klicken (kein Doppelklick), oder die Taste F2 betätigen. Einige Informationen können dabei nicht geändert werden - die sind einfach fest wie z.B. die Bitrate.
    Abbruch geht über die ESC-Taste, bestätigen mit der Enter-Taste.
    Bei den Bewertungen reicht ein Klick für das Setzen der neuen Bewertung.
  • In der Übersicht neben der Tabellenansicht (dort, wo Sie auch die erweiterten Tags ändern können). Einfach auf den Interpreten, Titel, Album klicken und die Information aktualisieren.
Sie können darüber auch die Informationen von Dateien in der Playlist bearbeiten. Wenn sich der Titel in der Playlist auch in der Medienbibliothek befindet, wird die Information auch in der Medienbibliothek aktualisiert.


Detailliertere Bearbeitung
Wenn Sie mehr wollen, können Sie das über das Kontextmenü "Eigenschaften"  machen. Dann öffnet sich ein weiteres Fenster, in dem Sie die ID3Tags bearbeiten können und z.B. auch Cover und Liedtexte einfügen können.


Liedtexte
Nemp kann zu einem Lied auch den Liedtext speichern, so dass man über die Suche auch ein Lied finden kann, das irgendwie so "in the town, where i was born" geht. Dieser Liedtext kann entweder von Hand eingegeben werden, oder automatisch aus dem Internet besorgt werden. Hierzu wird die Seite http://lyrics.wikia.com/ benutzt.
Das funktioniert für Sie genauso wie die Suche nach erweiterten Tags bei LastFM, auch wenn hier keine API benutzt wird (war mal so, wurde aber auf Druck der Plattenindustrie abgeschaltet, weil durch den automatischen Abruf der Songtexte ungefähr 42 Fantistillionen Umsatz verloren gehen und die Weltwirtschaft zugrunde geht), sodass der Liedtext etwas mühselig aus der Seite herausgepopelt werden muss. Aber das erledigt Nemp für Sie - zumindest solange das Seitendesign nicht allzusehr geändert wird.
Markieren Sie einfach ein paar Dateien und wählen über das Kontextmenü "Lyrics holen".


Aktualisieren der Daten
Wenn Sie einzelne Dateien mit anderen Programmen geändert haben, dann können Sie ein neues Einlesen für diesse Dateien erreichen.
Wählen Sie dazu über das Kontextmenü "Markierte aktualisieren".


Löschen von Dateien
Zum Löschen von Dateien aus der Medienbibliothek wählen Sie "Markierte Dateien aus der Bibliothek entfernen". Damit werden die Dateien aus der Bibliothek gelöscht, aber nicht von der Festplatte.
